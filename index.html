Lua-resty-socks5

Access .onion sites without Tor Browser

Download this project as a .zip file Download this project as a tar.gz file

This site is reverse proxy for Tor's hidden services.

To access hidden service using onion.gq, append .gq to its name. Example: access hidden service kpvzxxbbraaigawj.onion as kpvzxxbbraaigawj.onion.gq.
